# IPL Auction Management System

A simple IPL Auction System developed in C using CSV file handling.

## Features
- Team management
- Player auction simulation
- CSV-based data storage
- Budget handling
- Player allocation system

## Technologies Used
- C Programming
- File Handling
- CSV Data Processing

## Files
- `auction.c` → Main source code
- `players.csv` → Player database
- `teams.csv` → Team database

## How to Run

Compile:
```bash
gcc auction.c -o auction
```

Run:
```bash
./auction
```
