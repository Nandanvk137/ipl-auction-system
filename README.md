# 📌 Overview

The **IPL Auction Management System** is a C-based mini project that simulates the player auction process used in the Indian Premier League (IPL).

The system manages:
- Team creation
- Player allocation
- Budget tracking
- Auction simulation
- CSV-based persistent storage

This project was developed to strengthen concepts related to:
- File Handling in C
- CSV Data Processing
- Structures and Arrays
- Console-based Application Design
- Data Management Logic

# ⚙️ How It Works

1. Teams are initialized with a fixed auction budget.
2. Player data is loaded from CSV files.
3. The auction process simulates bidding and allocation.
4. Budgets are updated dynamically after purchases.
5. Final team compositions are stored and displayed.

# ▶️ How to Run

